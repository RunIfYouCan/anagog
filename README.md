# Anagog
## Structure
- `components` - Project components
- `containers` - Project containers
- `api`        - mock Api
- `redux`      - Redux actions\reducers
- `utils`      - Helpers
- `constants`  - Constants

## Build
#### Requirements:
- nodejs

Install dependencies:

```
npm install
```

Enable scripts:

#Start dev server
```
npm run start
```
#Create production build
```
npm run build
```

## Tests

#Run jest
```
npm run test
```

#Run tslint
```
npm run lint
```
